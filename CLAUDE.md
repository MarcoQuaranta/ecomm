# Istruzioni Progetto

## Landing Maker

Quando l'utente dice "vai in landing-maker", "landing maker", "apri landing maker",
"crea landing" o qualsiasi variazione simile:

1. Leggi IMMEDIATAMENTE il file `src/app/landing-maker/index.txt`
2. Segui le istruzioni passo per passo, partendo dalla Fase 0
3. Ogni fase ha un file di dettaglio separato (`fase1-analisi-copy.txt`, ecc.) - leggilo quando arrivi a quella fase
4. NON procedere alla fase successiva senza OK esplicito dell'utente
5. Usa la checklist (`checklist.md`) per verificare il completamento di ogni fase
